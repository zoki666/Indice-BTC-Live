💡 Guía de Estrategia Macro

🧭 Cómo leer este índice

El Índice BTC-Live va de 0 (techo de mercado) a 100 (suelo de mercado). La regla es simple:

  • ▲ Cuanto más alto (75-100) → zona de COMPRA.
    El mercado está barato, hay miedo, capitulación o infravaloración. Es el momento de acumular BTC con visión a largo plazo.
  • ▼ Cuanto más bajo (0-25) → zona de VENTA.
    El mercado está caro, hay euforia, codicia o sobrevaloración. Es el momento de tomar ganancias o proteger el capital.
  • ● Zona media (26-74) → NEUTRAL.
    No hay señales extremas en ninguna dirección. Se puede mantener la estrategia actual sin cambios drásticos.

💡 Cada indicador individual (C, F, R, E, K, T, U, V) también sigue esta misma lógica de 0 a 10, y el índice final es un promedio ponderado que da más peso a los indicadores con mejor historial predictivo.

📈 Cómo usar esta herramienta

No es un sistema de trading intradía, sino una guía para ciclos largos (semanas o meses). La forma más efectiva de usarla:

  • ✅ Cuando el índice entra en zona verde (≥ 75), programa compras periódicas (DCA) semanales o quincenales.
  • ⏸️ Cuando está en zona naranja (26-74), mantén tu posición.
  • ⚠️ Cuando entra en zona roja (≤ 25), deja de comprar y considera vender una parte (25-50%).
  • 🔄 El ciclo se repite: acumulas en verde, esperas en naranja, vendes en rojo.

🟢 Zona de Acumulación DCA (Índice ≥ 75)

Es el momento idóneo para comprar. El mercado está en una fase de miedo, infravaloración o capitulación.

  • ✔️ Programa compras automáticas periódicas (DCA).
  • ✔️ Aumenta ligeramente el tamaño de compra si el índice supera 85-90.
  • ✔️ Ignora el ruido mediático: las noticias serán negativas.
  • ✔️ Piensa a 6-18 meses vista.

🔴 Zona de Distribución / Venta (Índice ≤ 25)

Es el momento de asegurar ganancias. El mercado está en una fase de euforia, codicia o sobrevaloración.

  • ⚠️ Deja de comprar inmediatamente.
  • 💰 Vende un 25-50% de tu posición.
  • 🛑 Sube tus stop-loss.
  • 🧊 Guarda liquidez para la próxima zona de acumulación.

⚠️ Descargo de responsabilidad: Este índice es una herramienta educativa basada en datos históricos y no constituye asesoramiento financiero.
